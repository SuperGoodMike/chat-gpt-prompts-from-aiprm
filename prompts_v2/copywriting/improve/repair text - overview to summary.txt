Lalu untuk masing masing bagian/variabel, lakukan hal hal berikut ini (ada 5 perintah).
1. terhadap TEXTOVERVIEW : Mohon  di bagian TEXTOVERVIEW ini jangan sebutkan jumlah hari apapun, agar masih relevan. Jangan menggunakan kata 'yang lalu'. Jika ada yang kurang jelas, perbaiki text nya. Untuk bagian TEXTOVERVIEW ini, cukup  1 <blockquote> saja, dan per barisnya dipisahkan oleh tanda ganti baris yang tidak terlihat di wordpress text editor. 2. terhadap TEXTABOUT : Perbaikilah kalimat dan susunan kata-nya, kamu boleh menambahkan kata lainnya agar text nya terasa alamiah ditulis manusia biasa. Tetapi tetap harus menggunakan sebagian besar kata kata kunci/penting dari text TEXTABOUT aslinya. Tujuannya untuk membuat text ini mengerti maksud / intensi / intention para pencari lowongan kerja di Google Search. Buatlah 2 kalimat lain sebagai pendukung untuk bagian TEXTABOUT ini dan dituliskan masih dalam satu bagian tag <i> tanpa ganti baris, tanpa tag <P>. 3.  terhadap TEXTJOBDESCRIPTION: seluruh text setelah "<h3>Job Description:</h3>" hingga sebelum bagian text "Summary:" adalah merupakan bagian text TEXTJOBDESCRIPTION. Atau dengan kata lain, TEXTJOBDESCRIPTION adalah isi dari pasangan html tag '<div class="">' hingga '</div>' JANGAN MALAS UNTUK MENGAMBIL SEMUA ISI TEXTJOBDESCRIPTION yang ada. Jangan menghilangkan sebuah kalimat apapun. Tolong jangan mengurangi isi panjang teks nya. Kamu hanya boleh memperbaiki kalimat yang kurang enak, atau salah tulis.. Jjumlah paragraph harus tetap sama. Jumlah list baik <UL> maupun <OL> harus tetap sama. Saya lebih menyukai model <OL>. Jumlah kata kata di bagian TEXTJOBDESCRIPTION ini harus menjadi bertambah lebih banyak. Kamu boleh menambahkan kata untuk membuat text tersebut menggambarkan pekerjaan nyata di bidangnya. Perbaikilah spellingnya. Perbaikilah kalimatnya jika kurang jelas.  Ulangi mencantumlan kembali struktur HTML bagian Introduksi Perusahaaan/ company profile, bagian Deskripsi Lowongan Kerja dan bagian Kualifikasi Pekerjaan. Ubah keyword yang terkandung di dalamnya terkait masalah keyword pekerjaan menjadi bold <b> atau <strong>. Kamu seolah olah ber-gaya sebagai Kepala Bagian Human Resource Department yang sudah sangat berpengalaman mencari calon karyawan berkualitas. Perbaikilah logika susunan kalimatnya. Perbaikilah syntax isi UL dan OL nya. Gunakan tag html untuk berganti baris, dan  numbered list bernomor urut. Hindarkan adanya text yang mengarah kepada alamat URL atau domain pihak lain selain domain saya sendiri (job.web.id). Pisahkan tanggapanmu untuk pertanyaan ke - 3 ini tersendiri, dalam html code block. Outputnya harus dalam bentuk html code block terpisah TETAPI TIDAK PERLU DITAMPILKAN. Berikan informasi tentang besarnya persentase perubahan jumlah karakter pada bagian TEXTJOBDESCRIPTION setelah proses pengeditan dibandingkan dengan jumlah karakter sebelum proses. Jika memungkinkan, sertakan perhitungan persentasenya. Dituliskan di luar html code block tanggapan!"


4.  terhadap TEXTSUMMARY :  Mohon di bagian  TEXTSUMMARY ini jangan sebutkan jumlah hari apapun.  Jangan menggunakan kata 'yang lalu'. Perbaikilah kalimat dan susunan kata-nya, , kamu boleh menambahkan kata lainnya agar text nya terasa alamiah ditulis manusia yang mengerti struktur kalimat yang baik, masuk akal, bertatabahasa baik, terpelajar, dan berbudaya tinggi. Tetapi tetap harus menggunakan sebagian besar kata kata kunci/penting dari text TEXTSUMMARY. Jangan sebutkan angka uang di bagian ini. Tujuannya untuk membuat text ini mengerti maksud / intensi / intention para pencari lowongan kerja di Google Search. Berikan gambaran tentang pekerjaannya, dan perusahaan apa? Kembangkanlah menjadi minimal 2 - 4 kalimat penjelas lainnya untuk bagian TEXTSUMMARY ini, dan dituliskan masih dalam satu bagian tag <DIV> tanpa ganti baris, tanpa tag <P>. Pisahkan tanggapanmu untuk pertanyaan ke - 4 tersendiri, dalam html code block. Outputnya harus dalam bentuk html code block terpisah.Hanya boleh ada 1 <div class="jobsummary"> saja, namun boleh berisi hingga 4 kalimat.  Contoh format akhirnya adalah: <!-- 4. TEXTSUMMARY -->
Summary:<br>
<div class="jobsummary">
  Bergabunglah dengan tim Wesh Wash Laundry sebagai Staff Laundry. Temukan pengalaman kerja yang menarik, bonus lembur, dan peluang untuk belajar lebih banyak.
</div>


5. terhadap TEXTJOBDESCRIPTION, "Buatlah deskripsi meta yang bersumber dari text TEXTJOBDESCRIPTION tersebut. Agar menjadi deskripsi yang detail namun terdengar mudah dan informatif untuk posisi pekerjaan di sebuah lowongan kerja. Gunakan kata kunci yang sering diketikkan para pencari kerja atau dipikirkan oleh  para pencari kerja yang melakukan pencarian di Google Search. Hindari penggunaan nama "kota/daerah/tempat atau geo location". Hindari kata "penunjuk tempat". Hindari kata ajakan. Hindari kata "di".  Hindari kata hubung "dan,dari,ke,serta,meskipun,juga". Hindari kata "apply". Berikan gambaran tentang pekerjaannya, dan perusahaan apa? Maksimal 120 huruf. Pisahkan tanggapanmu untuk pertanyaan ke - 5 tersendiri, dalam html code block. Outputnya harus dalam bentuk html code block terpisah. Contoh format akhirnya adalah: <!-- 5. Deskripsi Meta -->
<meta name="description" content="
Lowongan Kerja Staff Laundry full-time di Wesh Wash Laundry, Tangerang.
" />
